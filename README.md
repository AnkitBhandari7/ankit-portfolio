## ✨ Features

| Feature | Description |
|---------|-------------|
| 🎮 **3D Character Model** | Interactive animated character rendered with Three.js & React Three Fiber |
| 🎯 **Smooth Scrolling** | GSAP ScrollSmoother with parallax effects and scroll-triggered animations |
| 📱 **Fully Responsive** | Optimized for all screen sizes with a mobile hamburger menu drawer |
| 🖱️ **Custom Cursor** | Dynamic cursor that reacts to interactive elements |
| 💼 **Project Showcase** | Carousel-style project viewer with image + description side-by-side |
| 🎖️ **Certifications** | AWS Academy certifications with direct verification links |
| 📬 **Contact Form** | Working email form via Web3Forms API with mailto fallback |
| ⚡ **Tech Stack Display** | Animated horizontal scrolling tech stack marquee |
| 🌙 **Dark Theme** | Premium dark UI with teal accent colors and glassmorphism effects |
| 📄 **Resume Download** | One-click PDF resume download |

## 🚀 Getting Started

### Prerequisites

- **Node.js** ≥ 18
- **npm** ≥ 9

### Installation

```bash
# Clone the repository
git clone https://github.com/AnkitBhandari7/ankit-portfolio.git

# Navigate to project directory
cd ankit-portfolio

# Install dependencies
npm install

# Start development server
npm run dev
```

The app will be available at `http://localhost:5173/`

### Build for Production

```bash
npm run build
```

## 📸 Sections

### 🏠 Hero & 3D Character
Interactive landing page with a 3D animated character model.

### 💼 Projects Showcase
Side-by-side layout: project info (description, tech stack, GitHub & Live Demo links) on the left, project image on the right. Navigate with arrow buttons.

### 🎖️ Certifications
AWS Academy certifications displayed in a 2×2 grid with direct verification links.

### 📬 Contact
Working contact form with email delivery via Web3Forms. Includes social links (GitHub, LinkedIn) and personal info.
